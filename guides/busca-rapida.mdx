# Busca rápida

Use esta página quando não souber exatamente onde procurar. Ela concentra os termos que mais ajudam a busca da documentação a encontrar o guia certo, o service correto ou um exemplo pronto.

## Perguntas prontas para a busca

Você pode pesquisar frases inteiras:

- `Como chamar SERVICE_OCR no /api/service-api?`
- `Qual payload para OCR de RG frente e verso?`
- `Como testar SERVICE_FACE_INDEX em HML?`
- `Don't have access to the service`
- `Por que result veio vazio?`
- `Qual service usar para risco de crédito PJ?`
- `Como consultar cartão CNPJ por OCR?`
- `Quais campos obrigatórios de SERVICE_CREDIT_RISK_COMPANY?`

## Se a busca não achar

1. Pesquise pelo alias completo, por exemplo `SERVICE_OCR`.
2. Pesquise pelo documento ou dado, por exemplo `CNH`, `CNPJ`, `score`, `telefone`.
3. Pesquise pelo problema, por exemplo `imagem ausente`, `result vazio`, `sem acesso`.
4. Abra o [índice de services](/guides/indice-de-services) e use `Ctrl + F`.
5. Se ainda não aparecer, confirme se o service já está documentado ou liberado para API.
