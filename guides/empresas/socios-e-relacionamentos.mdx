# Sócios e relacionamentos

Esta categoria reúne consultas para entender quem está por trás de uma empresa,
quais vínculos existem entre pessoas e CNPJs, e quais riscos aparecem no entorno
societário. Ela é útil para onboarding PJ, análise de beneficiário final, KYC de
sócios, avaliação de relacionamentos econômicos e investigação de vínculos
relevantes.

Todos os serviços desta categoria são consumidos pelo endpoint:

```http
POST /api/service-api
```

## Quando usar

Use esta categoria quando precisar:

- identificar proprietários, representantes e vínculos de uma empresa;
- mapear pessoas físicas relacionadas ao CNPJ em primeiro nível;
- avaliar processos judiciais associados aos sócios;
- consultar indicadores de PEP, sanções e histórico de compliance dos sócios.

## Serviços disponíveis

| Serviço | Código `service` | Uso principal |
| --- | --- | --- |
| Relacionamentos de uma empresa | `SERVICE_COMPANY_RELATIONSHIP_BIGDATACORP` | Retorna vínculos da empresa, incluindo proprietários, empregados, empresas possuídas e relacionamentos com pessoas |
| Sócios na Receita Federal | `SERVICE_COMPANY_RFB_OWNERS_BIGDATACORP` | Retorna dados cadastrais dos sócios identificados na Receita Federal |
| Sócios de primeiro nível | `SERVICE_FIRST_LEVEL_PARTNER_BIGDATACORP` | Resume círculos de pessoas físicas relacionadas ao CNPJ, com métricas de idade, renda, localização, processos e distribuições |
| Processos jurídicos dos sócios | `SERVICE_JURIDICAL_PROCESSES_PJ_OWNERS_BIGDATACORP` | Consulta processos associados aos sócios e agrupa indicadores por documento |
| KYC e compliance dos sócios | `SERVICE_COMPANY_KYC_OWNERS_BIGDATACORP` | Consulta exposição a PEP, sanções atuais e históricas, e ocorrências relevantes no quadro societário |
| Doações eleitorais dos sócios | `SERVICE_OWNERS_ELECTORAL_DONORS_CNPJ_BIGDATACORP` | Consulta doações eleitorais realizadas por sócios vinculados ao CNPJ |

## Exemplo rápido

Consulta de relacionamentos de uma empresa:

```bash
curl --location '{base_url}/api/service-api' \
  --header 'Authorization: Bearer {jwt_token}' \
  --header 'Content-Type: application/json' \
  --data '{
    "service": "SERVICE_COMPANY_RELATIONSHIP_BIGDATACORP",
    "cnpj": "cnpj"
  }'
```

Payload:

```json
{
  "service": "SERVICE_COMPANY_RELATIONSHIP_BIGDATACORP",
  "cnpj": "cnpj"
}
```

Resposta esperada:

```json
{
  "result": {
    "cnpj": "99244297000106",
    "relationshipTotal": 1,
    "totalOwners": 1,
    "totalEmployees": 1,
    "companyRelationships": [
      {
        "cpf": "77272722314",
        "name": "NOME DO SÓCIO",
        "relationshipName": "SOCIO-ADMINISTRADOR",
        "origin": "RECEITA FEDERAL"
      }
    ]
  },
  "status": {
    "code": 200,
    "message": "Success"
  }
}
```

## Como combinar os serviços

Uma análise PJ costuma começar com `SERVICE_COMPANY_RELATIONSHIP_BIGDATACORP` para entender a
estrutura da empresa. Em seguida, `SERVICE_FIRST_LEVEL_PARTNER_BIGDATACORP` ajuda a resumir o
entorno societário. Quando houver necessidade de aprofundar risco, use
`SERVICE_JURIDICAL_PROCESSES_PJ_OWNERS_BIGDATACORP` e `SERVICE_COMPANY_KYC_OWNERS_BIGDATACORP`.

Para campos, exemplos completos e responses, consulte a
[API Reference](/api-reference/autorização/gerar-token-de-api) ou o
[Catálogo técnico de Pessoa Jurídica](/guides/servicos-pessoa-juridica).
