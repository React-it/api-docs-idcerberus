# Matriz de serviços

Esta matriz ajuda a localizar rapidamente qual serviço usar antes de abrir o
payload completo no API Reference.

<Tip>
  Todos os itens abaixo são executados por `POST /api/service-api`, exceto os
  endpoints de autenticação, onboarding e customers. No endpoint central, o campo
  `service` escolhe o produto.
</Tip>

## Pessoa física

| Objetivo | Serviço | Entrada principal | Quando usar |
| --- | --- | --- | --- |
| Enriquecer dados cadastrais | `SERVICE_PERSON_DATA_ENRICHMENT_BIGDATACORP` | `cpf` | Obter dados básicos de identificação |
| Consultar CPF na Receita | `SERVICE_RFB_PF_BIGDATACORP` | `cpf` | Validar situação cadastral do CPF |
| Consultar CPF na Receita on-demand | `SERVICE_RFB_PF_ON_DEMAND_BIGDATACORP` | `cpf` | Buscar dados cadastrais no momento da consulta |
| Consolidar modelagem de dados PF | `SERVICE_PERSON_DATA_MODELING_BIGDATACORP` | `cpf` | Reunir sinais cadastrais, contatos, vínculos e risco em texto consolidado |
| Gerar resumo analítico de pessoa | `SERVICE_PERSON_AI_PROMPT_OPENAI` | `cpf` | Apoiar leitura de dados consolidados com uma resposta textual |
| Extrair dados de documento | `SERVICE_OCR_BIGDATACORP` | `image1`, `image2` ou URLs | Ler CNH, RG, RNE, CPF, CRLV e outros documentos |
| Extrair RG ou CNH via OCR React | `SERVICE_OCR` | `documentType`, `image1`, `image2` | Ler RG ou CNH e retornar dados limpos no `result` |
| Ler documento de emancipação | `SERVICE_OCR_EMANCIPATION` | `image1` | Extrair texto e dados possíveis de documento variável |
| Ler comprovante de endereço | `SERVICE_OCR_PROOF_OF_ADDRESS` | `image1` | Extrair nome, endereço, datas e valores quando disponíveis |
| Buscar face na base | `SERVICE_FACE_INDEX` | `image1` | Procurar rosto em base indexada e retornar CPF/similaridade quando houver |
| Comparar faces | `SERVICE_FACE_MATCH_BIGDATACORP` | `image1`, `image2` | Comparar selfie com documento ou outra imagem |
| Validar presença real | `SERVICE_LIVENESS_2D_FACETEC` | `image1` | Verificar se a selfie aparenta ser de pessoa real |
| Validar CNH no DataValid | `SERVICE_DATAVALID_CNH_SERPRO` | `cpf`, `image1` | Validar dados de CNH com apoio do DataValid/Serpro |
| Consultar PEP | `SERVICE_PEP` | `cpf` | Avaliar exposição política |
| Consultar KYC/compliance PF | `SERVICE_PERSON_KYC_BIGDATACORP` | `cpf` | Análise mais ampla de compliance |
| Consultar risco financeiro | `SERVICE_FINANCIAL_RISK_SCORE_BIGDATACORP` | `cpf`, `birthDate` | Obter score e leitura resumida de risco financeiro |
| Consultar score de crédito | `SERVICE_CREDIT_SCORE` | `cpf` | Obter score de crédito via Assertiva |
| Consultar score Quantum | `SERVICE_DEFAULT_RISK_SCORE_QUANTUM` | `cpf` | Avaliar risco de inadimplência |
| Consultar dados demográficos | `SERVICE_DEMOGRAPHIC_DATA_CPF` | `cpf`, `birthDate` | Obter indicadores demográficos do CPF |
| Consultar indicadores de atividades | `SERVICE_ACTIVITIES_INDICATORS` | `cpf` | Obter sinais profissionais e de atividade |
| Consultar domínios PF | `SERVICE_DOMAINS_CPF` | `cpf` | Buscar domínios e sinais digitais associados ao CPF |
| Consultar prêmios e certificações | `SERVICE_AWARDS_AND_CERTIFICATIONS_CPF` | `cpf` | Obter prêmios/certificações encontrados |
| Consultar benefícios sociais familiares | `SERVICE_FAMILY_SOCIAL_BENEFITS` | `cpf` | Verificar benefícios sociais familiares |
| Consultar benefícios sociais estendidos | `SERVICE_SOCIAL_ASSISTANCE_EXTENDED` | `cpf` | Verificar benefícios sociais estendidos |
| Consultar processos | `SERVICE_JURIDICAL_PROCESSES_BIGDATACORP` | `cpf` | Levantar processos jurídicos e administrativos |
| Consultar histórico profissional | `SERVICE_PROFESSIONAL_HISTORY_BIGDATACORP` | `cpf` | Recuperar vínculos profissionais |
| Consultar histórico profissional do titular | `SERVICE_PROFESSIONAL_HISTORY_OWNER_ONLY_BIGDATACORP` | `cpf`, `birthDate` | Recuperar vínculos em que a pessoa aparece como titular ou sócia |
| Consultar dados financeiros e endereços | `SERVICE_PF_FINANCIAL_AND_ADDRESS_BIGDATACORP` | `cpf`, `birthDate` | Combinar dados cadastrais, endereços e informações financeiras |
| Consultar propensão a apostas online | `SEVICE_ONLINE_BETTING_PROPENSITY_BIGDATACORP` | `cpf` | Avaliar sinais de propensão a apostas online |
| Consultar histórico de telefones | `SERVICE_PHONE_HISTORY_BIGDATACORP` | `cpf`, `birthDate`, `limit` | Recuperar telefones associados ao CPF |
| Consultar histórico de e-mails | `SERVICE_EMAILS_EXTENDED_BIGDATACORP` | `cpf`, `limit` | Recuperar e-mails associados ao CPF |
| Consultar pessoas relacionadas | `SERVICE_RELATED_PEOPLE_BIGDATACORP` | `cpf`, `birthDate` | Mapear vínculos pessoais associados ao CPF |
| Validar CPF com telefone | `SERVICE_CPF_PHONE_VALIDATION_BIGDATACORP` | `cpf`, `phone` | Conferir associação entre CPF e telefone |
| Validar CPF com endereço | `SERVICE_CPF_ADDRESS_VALIDATION_BIGDATACORP` | `cpf`, `zipcode`, `numberAddress` | Conferir associação entre CPF e endereço |

## Fluxos fora do service-api

| Objetivo | Endpoint | Quando usar |
| --- | --- | --- |
| Gerar token JWT | `POST /api/token-generate` | Antes de chamadas protegidas |
| Gerar token de onboarding | `POST /api/token-history-onboarding` | Antes de iniciar integração via SDK |
| Consultar onboarding | `GET /api/onboarding/report/{tokenOnboarding}` | Para recuperar resultado estruturado em JSON |
| Baixar relatório de onboarding | `GET /api/history-onboarding-report/{tokenOnboarding}` | Para obter o PDF final |
| Consultar cliente | `GET /api/customer` | Para localizar cliente por CPF ou CNPJ |
| Alterar status de cliente | `POST /api/changeStatusOfCustomer` | Para ativar ou desativar cliente |
