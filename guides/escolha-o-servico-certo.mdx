# Escolha o serviço certo

Use esta página para sair de uma necessidade de negócio e chegar nos códigos
`service` que devem ser usados em `POST /api/service-api`.

## Atalhos por necessidade

| Se você quer... | Use este serviço |
| --- | --- |
| Consultar dados cadastrais de CPF | `service_person_data_enrichment` |
| Verificar CPF na Receita Federal | `service_rfb_pf` |
| Extrair dados de documento | `service_ocr` |
| Comparar selfie e documento | `service_face_match` |
| Validar prova de vida | `service_liveness_2d` |
| Consultar PEP | `service_pep` |
| Consultar dados cadastrais de CNPJ | `service_corporate_data_enrichment` |
| Verificar CNPJ na Receita Federal | `service_rfb_pj` |
| Consultar sócios ou vínculos da empresa | `service_company_relationship`, `service_first_level_partner` |
| Consultar débitos de empresa | `SERVICE_ACTIVE_DEBT_PJ` |

## Validar identidade de uma pessoa

| Necessidade | Serviços indicados | Quando usar |
| --- | --- | --- |
| Confirmar dados cadastrais do CPF | `service_person_data_enrichment`, `service_rfb_pf` | Validar nome, nascimento, situação cadastral e origem Receita Federal |
| Extrair dados de documento | `service_ocr` | Ler CNH, RG, RNE, CPF, passaporte, CRLV e outros documentos suportados |
| Comparar selfie e documento | `service_face_match` | Validar se duas imagens pertencem a mesma pessoa |
| Validar prova de vida | `service_liveness_2d` | Reduzir risco de foto, tela ou tentativa de fraude simples |
| Consultar biometria governamental | `service_biometric_score` | Obter score de similaridade com bases governamentais |
| Fazer análise documental completa | `service_digital_documentoscopy` | Extrair dados e combinar documento com selfie em um fluxo único |

## Montar um onboarding completo

| Etapa | Serviços ou endpoints |
| --- | --- |
| Criar jornada via SDK | `POST /api/token-history-onboarding` |
| Capturar documento e selfie | SDK ou Cliente Web |
| Consultar resultado | `GET /api/onboarding/report/{tokenOnboarding}` |
| Baixar evidências | `GET /api/history-onboarding-report/{tokenOnboarding}` |
| Complementar com consultas avulsas | `service_rfb_pf`, `service_pep`, `service_face_match`, `service_biometric_score` |

## Analisar uma empresa

| Necessidade | Serviços indicados |
| --- | --- |
| Dados cadastrais de CNPJ | `service_corporate_data_enrichment`, `service_rfb_pj` |
| Inscrição estadual e operação | `service_sintegra_consultation` |

## Combinações comuns

| Fluxo | Combinação recomendada |
| --- | --- |
| Cadastro simples de CPF | `service_rfb_pf` + `service_person_data_enrichment` |
| Onboarding com documento | `service_ocr` + `service_face_match` + `service_liveness_2d` |
| KYC pessoa física | `service_pep` + certidões + débitos + score de risco |
| Cadastro de empresa | `service_rfb_pj` + `service_corporate_data_enrichment` + `service_sintegra_consultation` |
| KYC empresarial | dados PJ + sócios + KYC dos sócios + débitos + protestos |
