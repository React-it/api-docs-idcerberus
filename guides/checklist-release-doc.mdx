# Checklist de release da documentação

Use este checklist antes de publicar uma nova versão da documentação.

## Antes de gerar

| Item | Status |
| --- | --- |
| Confirmei quais services foram alterados ou adicionados | Pendente |
| Conferi se os novos services existem via API, SDK, backoffice ou onboarding | Pendente |
| Evitei trazer itens encontrados apenas em CSV antigo sem confirmação | Pendente |
| Revisei nomes, acentuação, pontuação e exemplos principais | Pendente |

## Comandos obrigatórios

Execute na raiz do projeto:

```bash
npm.cmd run generate:llms
npm.cmd run map:api-doc
npm.cmd run check:text
npm.cmd run validate
```

O resultado esperado é:

| Comando | Resultado esperado |
| --- | --- |
| `generate:llms` | LLMs, catálogo, índice e cobertura regenerados |
| `map:api-doc` | Mapeamento de API real atualizado |
| `check:text` | Sem mojibake, links quebrados ou exemplos desalinhados |
| `validate` | `success build validation passed` |

## Conferência visual

| Item | Status |
| --- | --- |
| Logo aparece no modo claro | Pendente |
| Logo aparece no modo escuro | Pendente |
| Favicon aparece no navegador | Pendente |
| Guia abre sem 404 | Pendente |
| API Reference abre e permanece na seção correta | Pendente |
| Busca encontra termos como CPF Receita, CNPJ Receita, OCR, KYC e FaceMatch | Pendente |

## Conferência de conteúdo

| Item | Status |
| --- | --- |
| Quickstart está executável por pessoa não técnica | Pendente |
| API Reference tem request e response para os serviços novos | Pendente |
| `services-catalog.json` foi regenerado | Pendente |
| `llms-api-real.txt` foi regenerado | Pendente |
| Changelog foi atualizado | Pendente |
| Não há token, secret, CPF real, CNPJ real ou imagem real nos exemplos | Pendente |

## Depois do deploy

| Item | Status |
| --- | --- |
| A URL publicada abre corretamente | Pendente |
| Navegação entre Guias e API Reference funciona | Pendente |
| Páginas internas não retornam 404 | Pendente |
| O arquivo `/llms.txt` abre pela URL publicada | Pendente |
| O arquivo `/llms-api-real.txt` abre pela URL publicada | Pendente |
