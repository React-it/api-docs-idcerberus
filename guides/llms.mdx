# Arquivos para LLMs

A documentação também publica arquivos em texto simples e JSON para uso por
LLMs, assistentes de código e ferramentas de automação.

Esses arquivos ajudam a consultar rapidamente guias, endpoints, services,
ambientes e exemplos da API idCerberus.

<Info>
Os links abaixo são arquivos estáticos. Ao clicar, o navegador abre o conteúdo
do arquivo em uma nova página. Para usar em uma ferramenta de IA ou automação,
copie a URL completa.
</Info>

## Arquivos disponíveis

| Arquivo | Quando usar |
| --- | --- |
| [`llms.txt`](https://react-it.github.io/api-docs-idcerberus/llms.txt) | Índice curto da documentação, com links principais e visão geral. |
| [`llms-small.txt`](https://react-it.github.io/api-docs-idcerberus/llms-small.txt) | Resumo operacional para integrações: ambientes, autenticação, `POST /api/service-api`, fluxos e services documentados. |
| [`llms-full.txt`](https://react-it.github.io/api-docs-idcerberus/llms-full.txt) | Conteúdo completo consolidado dos guias, API Reference e OpenAPI. |
| [`llms-api-reference.txt`](https://react-it.github.io/api-docs-idcerberus/llms-api-reference.txt) | Versão operacional do API Reference, com services e exemplos de curl prontos para consulta por IA. |
| [`services-catalog.json`](https://react-it.github.io/api-docs-idcerberus/services-catalog.json) | Catálogo estruturado em JSON para automações, validadores e agentes. |
| [`examples/auth.hml.curl`](https://react-it.github.io/api-docs-idcerberus/examples/auth.hml.curl) | Exemplo pronto de autenticação em homologação. |

## Uso recomendado

Para dúvidas rápidas sobre a estrutura da documentação, use:

```txt
https://react-it.github.io/api-docs-idcerberus/llms.txt
```

Para pedir ajuda de implementação, geração de `curl`, escolha de service ou
explicação de fluxo, use:

```txt
https://react-it.github.io/api-docs-idcerberus/llms-small.txt
```

Para gerar exemplos de request a partir dos services documentados, use:

```txt
https://react-it.github.io/api-docs-idcerberus/llms-api-reference.txt
```

Para análises mais completas ou perguntas que dependem de todo o conteúdo da
documentação, use:

```txt
https://react-it.github.io/api-docs-idcerberus/llms-full.txt
```

Para automações que precisam de dados estruturados, use:

```txt
https://react-it.github.io/api-docs-idcerberus/services-catalog.json
```

## Exemplo de prompt

```txt
Use esta documentação como fonte principal:
https://react-it.github.io/api-docs-idcerberus/llms-small.txt

Preciso integrar a API idCerberus para consultar dados de CPF.

Não invente services que não estejam na documentação.
```
