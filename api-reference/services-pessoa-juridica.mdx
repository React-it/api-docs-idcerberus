# Services de pessoa jurídica

Catálogo explícito dos services de pessoa jurídica disponíveis via API, com campos esperados e exemplos de request.

<Info>
Todos os services abaixo usam o mesmo endpoint: `POST /api/service-api`. O produto executado é definido pelo campo `service` no body da requisição.
</Info>

## Como ler esta referência

- **Nome**: nome funcional do produto.
- **Service**: valor exato que deve ser enviado no campo `service`.
- **Campos principais**: campos esperados no body além de `service`.
- **Exemplo**: cada item traz JSON e curl de homologação prontos para copiar e adaptar.

<Accordion title="Certidão negativa de protesto">

**Service:** `service_protest_clearance_certificate_pj`

**Endpoint:** `POST /api/service-api`

**Campos principais:** `cnpj`

### Body

```json
{
  "service": "service_protest_clearance_certificate_pj",
  "cnpj": "cnpj"
}
```

### curl de homologação

```bash
curl --location 'https://backoffice-hml.idcerberus.com/api/service-api' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {jwt_token}' \
--data '{
  "service": "service_protest_clearance_certificate_pj",
  "cnpj": "cnpj"
}'
```

### Retorno
