# Services por caso de uso

Use esta página quando souber o objetivo da integração, mas ainda não souber qual `service` chamar.

<Info>
Depois de escolher o service, abra o catálogo de pessoa física ou pessoa jurídica para copiar o request completo.
</Info>

## Como escolher

1. Comece pelo objetivo da consulta.
2. Copie o `service` indicado.
3. Abra o catálogo de pessoa física ou jurídica para ver payload e retorno.
4. Se for OCR, confira o guia de imagem antes de testar.
5. Se quiser um fluxo pronto, use [Receitas prontas](/guides/receitas-prontas).
