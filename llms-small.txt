# idCerberus API Docs - resumo operacional para LLM

Use este arquivo quando precisar de contexto rapido para integrar com a API idCerberus.

## Regras para assistentes de IA

- Use a documentacao como fonte principal e nao invente endpoints, parametros ou services.
- Para consultas externas, use `POST /api/service-api` e selecione o produto pelo campo `service`.
- Use `Authorization: Bearer {jwt_token}` em chamadas protegidas.
- Use homologacao para testes e producao somente quando o usuario pedir explicitamente.
- Nunca exponha tokens, secrets, CPFs, CNPJs ou imagens reais em exemplos.
- Se um service nao aparecer no catalogo, informe que ele precisa ser confirmado antes de documentar ou integrar.

## Ambientes

- Homologacao: `https://backoffice-hml.idcerberus.com`
- Producao: `https://backoffice.idcerberus.com`

## Autenticacao

- Gere token em `POST /api/token-generate` com `client` e `secret`.
- Envie o token nas chamadas protegidas com `Authorization: Bearer {jwt_token}`.
- Quando expirar, gere um novo token.

## Endpoint principal

- Use `POST /api/service-api` para consultas de dados, risco, compliance, biometria e enriquecimento.
- O campo `service` define qual produto sera executado.
- Os demais campos variam conforme o servico escolhido.

## Fluxos principais

- [Quickstart](https://react-it.github.io/api-docs-idcerberus/guides/quickstart): Faça sua primeira chamada na API idCerberus com Postman, Windows, macOS ou Linux
- [Autenticação](https://react-it.github.io/api-docs-idcerberus/guides/autenticacao): Gere tokens JWT e autentique chamadas protegidas da API idCerberus
- [Primeira consulta CPF](https://react-it.github.io/api-docs-idcerberus/guides/primeira-consulta-cpf): Execute uma consulta simples de pessoa física do início ao fim
- [Primeira consulta CNPJ](https://react-it.github.io/api-docs-idcerberus/guides/primeira-consulta-cnpj): Execute uma consulta simples de pessoa jurídica do início ao fim
- [Onboarding via SDK](https://react-it.github.io/api-docs-idcerberus/guides/onboarding-sdk): Gere TokenOnboarding, acompanhe o status do cadastro e baixe o relatório final
- [Matriz de serviços](https://react-it.github.io/api-docs-idcerberus/guides/matriz-de-servicos): Visão resumida dos principais serviços por tipo de documento, entrada e momento de uso
